Notice that in the following exercises, you are logging into Linkurious before opening the visualization.
The domain of the current page is academy.linkurio.us and the domain of the Linkurious server is crunchbase.linkurio.us.

In order to enable login to the Linkurious server from your custom application, we recommend that you host your Linkurious server at a domain like linkurious.example.com and your custom application at app.example.com.
At the same time, set the server.cookieDomain to .example.com (notice the leading .) in the Linkurious configuration file. This allows the Linkurious session cookie to be shared across all *.domain.com sub-domains.
