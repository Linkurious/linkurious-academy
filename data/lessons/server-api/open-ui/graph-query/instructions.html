The user interface can be opened with the result of a pattern query. You can write the query in Cypher if the data-source is a Neo4j database, or write in Gremlin if it is a TitanDB database.

Edit the query string in Cypher to get 5 random nodes, then validate your code.

Read the Cypher Refcard or the full Cypher documentation in the Neo4j Manual to learn more.
