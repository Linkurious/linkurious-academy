In this tutorial you will learn how to create a mini web interface for users to search nodes, select a result, and to display its connections in the same page.

We will first use the Search API of the Linkurious server to provide instant results. Once the end user selects a result, we will query the Graph API get its information and its neighborhood, i.e. its connected nodes and the relationships between those nodes. We will then display the graph with the linkurious.js Javascript library. As an exercise bonus, we will finally allow users to expand a node's connections by double-click on it in the visualization.
