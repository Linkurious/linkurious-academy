Welcome to the tutorial of the Linkurious API server! You will learn how to query the REST API of the Linkurious server through a series of exercises. Each exercise contains a few lines of Javascript code to write.

Click on the "Validate" button below the code area to validate your code. You can always reveal the solution with a click on the "Cheat" button, or click on the "Reset" button to re-initialize the code. Once your code is valid, click on the "Next exercise" button to move to the next step, or pick any exercice on the left menu.

You can ask for help at support@linkurio.us.
